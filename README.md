## Portfolio Website
A personal portfolio website built with React.js and Bootstrap. This single-page web application features a clean, scrollable design, allowing users to navigate seamlessly through sections including Home, Projects, and Contact.

## 📄 Features

Single-Page Scrollable Design: Navigate between sections by scrolling or clicking on navigation links.
Home Section: An introduction with a profile image, a brief bio, and links to GitHub, LinkedIn, and email.
Projects Section: A showcase of projects, each with descriptions, technologies used, and links to GitHub repositories.
Contact Section: Contact information and a form for users to reach out directly from the website.

## 🚀 Technologies Used
React.js: Framework for building the single-page application.
Bootstrap: For responsive layout and styling.
JavaScript, HTML, CSS: Core web technologies.
